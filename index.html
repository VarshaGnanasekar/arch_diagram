Phase 4 — Graph neural network
train_gnn.py · GraphSAGE 2-layer · 3,000 nodes · 545,656 edges (sampled 100k)
Graph construction
Nodes = unique VPAs (3,000) · Edges = transactions · 100k sampled from 545k · Node features = 12 aggregated stats
91.1% nodes are fraud senders
GraphSAGE 2-layer
Layer 1: 12→64 · Layer 2: 64→32 · BatchNorm + ReLU + Dropout · 10,177 params · early stop ep 52
Best val AUC-PR: 0.9996
Transaction score mapping
txn_score = max(sender_prob, receiver_prob) · val AUC-PR drops to 0.020 at txn level → GNN gets 0% ensemble weight
Node AUC-PR 0.999 · Txn AUC-PR 0.020
⚠ GNN scored 0% weight in ensemble. Node-level accuracy is 99.9% but the node→transaction score mapping loses signal (91.1% of nodes are fraud senders so all transactions look risky). Would contribute more in production with millions of VPAs.
